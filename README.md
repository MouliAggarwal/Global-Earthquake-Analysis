# 🌍 Global Earthquake Analysis Dashboard
## 📌 Project Overview

This project involves:

- Cleaning and preprocessing earthquake data using Google Colab.
- Enriching the dataset by identifying **countries** and **continents** from latitude and longitude coordinates.
- Transforming data in Microsoft Excel for better readability.
- Building an interactive Power BI dashboard to visualize earthquake trends and patterns.

---

## 📂 Dataset Features

The dataset includes the following attributes:

- Magnitude
- Date & Time
- Community Determined Intensity (CDI)
- Alert Level
- Tsunami Indicator
- Significance Score
- Magnitude Type
- Depth
- Latitude
- Longitude
- Location
- Country
- Continent

---

## 🛠️ Tools & Technologies

- **Google Colab** – Data preprocessing and transformation
- **Microsoft Excel** – Data cleaning and formatting
- **Power BI** – Dashboard creation and data visualization

---

## 📊 Dashboard Highlights

The dashboard provides insights through:

- 🌍 Global Distribution of Earthquakes
- 🌎 Earthquake Count by Continent
- 📏 Average Earthquake Depth by Continent
- 📈 Relationship Between Depth and Magnitude
- 🌊 Earthquakes with and without Tsunamis
- 🚨 Earthquake Alert Level Distribution

### Key Performance Indicators (KPIs)

- Average Magnitude
- Maximum Magnitude
- Average Depth
- Total Tsunami Events
- Countries 

---

## 📈 Key Insights

- Oceania, Asia, and South America recorded a high number of earthquakes.
- Earthquake depths vary significantly across continents.
- Most recorded earthquakes did not result in tsunami warnings.
- Higher magnitude earthquakes are not always the deepest.
- The dashboard enables interactive exploration through geographic and statistical visualizations.
